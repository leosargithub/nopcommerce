# nopcommerce
Ecommerce Site Automation 
nopCommerce E-Commerce Automation (Pytest Framework)

This project contains automated test scripts for the nopCommerce eCommerce website using the Pytest framework. The automation suite is designed to validate core functionalities such as user registration, login, product search, cart operations, checkout process, and order confirmation.

The framework follows best practices including the Page Object Model (POM) design pattern for better maintainability and scalability. It integrates Selenium WebDriver for browser automation and generates detailed test reports for execution results.

Key Features
Automated end-to-end test scenarios
Page Object Model (POM) structure
Pytest fixtures for setup and teardown
Cross-browser testing support
HTML test reporting
Easy test execution via command line
Tech Stack
Python
Pytest
Selenium WebDriver
WebDriver Manager
